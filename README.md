# Say Cheese 🧀📷

Say hello to **Say Cheese** – your own virtual photo booth built using **Next.js**, **Tailwind CSS**, and pure vibe-coding energy ⚡

Take 3 fun snapshots with live filters, countdown animation, and download them as a stacked **photo strip** – just like the good old photo booths.

## 🎬 Live Preview

🚀 [Try it out here](https://say-cheese-dhruv0050.vercel.app)

## 📸 Features

- 🎞️ Capture 3 sequential photos from your webcam
- 🎨 Real-time filters: grayscale, sepia, blur, and more
- ⏱️ Fun countdown animation before each click
- 🖼️ Auto-generated vertical photo strip layout
- 📥 Download the strip as an image (HTML2Canvas-powered)
- 🔁 Option to retake the strip anytime

## 🧠 Tech Stack

- **Frontend:** Next.js 14, Tailwind CSS
- **Camera:** `getUserMedia()` Web API
- **Filter Effects:** CSS filters
- **Image Export:** `html2canvas`
- **UX Animations:** Custom countdown overlay

## 🚀 How to Run Locally

```bash
git clone https://github.com/dhruv0050/Say-Cheese.git
cd Say-Cheese
npm install
npm run dev
````

Then visit `http://localhost:3000` in your browser.

> Make sure your browser allows webcam access.
